# Skyceil

A clean and minimalistic JSDoc theme

# [Documentation](https://skyceil.zyrouge.gq)

## Install
```bash
$ npm install jsdoc-skyceil
```

## Usage
```bash
$ jsdoc ./ -t /path/to/skyceil
```

## Clone
```bash
$ git clone https://github.com/zyrouge/jsdoc-skyceil
```
